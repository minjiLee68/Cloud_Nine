# Cloud Nine (더할 나위 없이 행복하다 / 행복의 절정)

### 로고 설명
- 사랑하는 학생들을 가르치고 학생들의 올바른 성장을 도와주고 바라보는 선생님들의 마음을 잘 대변하는 단어라고 생각해 앱 이름으로 선정했습니다.
- 학생들을 나타내고 특정 학생을 선택(?)한 느낌을 주는 아이콘을 넣었습니다.
### 앱 한줄 설명 
- 모든 선생님들을 위한 출석부 앱

## - App 이름
    Cloud Nine / 클라우드 나인

## - App 간단 소개
    Cloud Nine은 학생들을 관리하는 선생님들을 위한 어플입니다. 학생들의 인적사항을 관리 하고 학생들의 특징들을 쓰면서 학생들을 더 관심있게 바라볼 수 있습니다.
    그리고 어플 사용자들을 나타낼 수 있는 프로필과 닉네임, 계정을 통해
    연결된 선생님들끼리 대화할 수 있고, ToDoList가 있는 Calendar를 통해
    스케줄 관리도 할 수 있습니다.
